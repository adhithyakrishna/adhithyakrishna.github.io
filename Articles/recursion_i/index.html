Recursion Learning

Recursion is bascially a function calling itself. Recursion has to be implemented carefully, otherwise the function might run indefinitely causing the program to crash.

There are two important properties to recursion,

1) Base case - a terminating scenario, which does not use recursion to produce answer.

2) Recurrence relation - reduces other cases towards the base case.

The difficulty of recursion problems solved in this articles are going to be in the increasing order.

Reverse string recursively

package com.learning.recursion;

public class RecursivePrintString {

	public static void printReverse(String val) {
		if (val == null)
			return;
		// the second parameter is to keep track of indexes
		reverseWords(val, 0);
	}

	public static void reverseWords(String val, int index) {
		// base condition
		if (index >= val.length()) {
			return;
		}

		//recurrance relation
		reverseWords(val, index + 1);
		System.out.println(val.charAt(index));
	}

	public static void main(String args[]) {
		String val = "helper";
		printReverse(val);
	}
}

Complexity

Time complexity: O(N)
Space complexity: O(N)

Reverse character array

package com.learning.recursion;

public class ReverseString {

	public static void reverseString(char[] s) {
		if (s.length == 0) {
			return;
		}
		int[] val = new int[1];
		reverseInPairs(s, 0, s.length-1);
	}

	//reverses the character array in place
	public static void reverseInPairs(char[] s, int left, int right) {
		if(left >= right) return;
		
		char temp = s[left];
		s[left] = s[right];
		s[right] = temp;
		
		left++;
		right--;
		
		reverseInPairs(s, left, right);
	}

	public static void main(String args[]) {
		char[] s = { 'h', 'e', 'l', 'l', 'o' };
		reverseString(s);

		for(char ch : s)
			System.out.println(ch);
	}
}

Complexity

Time complexity: O(N)
Space complexity: O(N)

Swap nodes in pairs [Linked list]

Given a linked list swap the nodes in pairs recursively

/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode() {}
 *     ListNode(int val) { this.val = val; }
 *     ListNode(int val, ListNode next) { this.val = val; this.next = next; }
 * }
 */
class Solution {
    public ListNode swapPairs(ListNode head) {
        
        if(head == null || head.next == null) {
            return head;
        }
        ListNode firstNode = head;
        ListNode secondNode = head.next;
        
        firstNode.next = swapPairs(secondNode.next);
        secondNode.next = firstNode;
        
        return secondNode;
    }
}

Complexity

Time complexity: O(N)
Space complexity: O(N)

Reverse a linked list [Linked list]

Given a linked list reverse the linked list recursively

/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode() {}
 *     ListNode(int val) { this.val = val; }
 *     ListNode(int val, ListNode next) { this.val = val; this.next = next; }
 * }
 */
class Solution {
    
	public ListNode reverseList(ListNode head) {
        return reversing(head, null);
    }

	public ListNode reversing(ListNode curr, ListNode prev)
	{
		if(curr == null) //we have reached the end
		{
			return prev; // prev would be the head of the node
		}
		//keep a copy of the next node
		ListNode next = curr.next;
		//assign previous value to the current's next
		curr.next = prev;		
		//recursively call the reversing, note the current for this iteration is the previous value and next copy we took is the current value
		reversing(next, curr);

	}
}

Complexity

Time complexity: O(N)
Space complexity: O(N)

Recursive pascal triangle

Find the nth row of the pascal triangle recursively.

class Solution {
    
    int[][] cache;
    
    public List<Integer> getRow(int rowIndex) {
        //cache is created for the entire matrix
        cache = new int[rowIndex+1][];
        
        for(int i=0; i<cache.length; i++)
        {
            cache[i] = new int[i+1];         
        }
        
        List<Integer> dataList = new ArrayList<Integer>(rowIndex);
        
        for(int i=0; i<rowIndex+1; i++) {
            dataList.add(recurse(rowIndex, i));    
        }
        
        return dataList;
    }
    
    public int recurse(int row, int col)
    {
        if(row == 0 || col == 0 || row == col) {
            return 1;
        }
        
        if(cache[row][col] == 0)
            cache[row][col] = recurse(row-1, col-1) + recurse(row - 1, col);
        
        return cache[row][col];
    }
    
}

Complexity

Time complexity: O(2^k)
Space complexity: O(n^2)
